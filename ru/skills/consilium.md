# consilium

Используй для high-stakes решений, где независимые мнения уменьшают blind spots.

Сформулируй decision, constraints и success criteria. Дай одинаковый prompt 2-4 независимым перспективам, не показывая ответы друг другу.

Синтез: решение, почему, rejected alternatives, confidence, risks, next verification step.
