# verifying

Используй при приёмке любой выполненной работы.

Verifier получает только спеку, артефакты/diff и DoD-команды. Возвращает `PASS`, `FAIL` или `UNVERIFIED` по каждому пункту с evidence.

Лестница провалов: 1-2 — тому же исполнителю с конкретными расхождениями; 3 — свежий исполнитель; 4 — blocked.
