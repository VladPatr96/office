# Трассировка Fable 5

Эта матрица сохраняет legacy principles внутри Office без привязки к домену или конкретной модели.

| Принцип | Где в Office |
|---|---|
| Простота побеждает | `office`, `writing-specs`, `next-step` |
| Агенту нужны инструменты | `registry.md`, `design-stack`, install scripts |
| Маленькие изменения и проверки | `writing-specs`, `reproduce`, `verifying` |
| Живая документация | `done-check`, `lesson`, `planerka` |
| Consent для внешних действий | `office`, `design-stack`, `CLAUDE.md` |
| Время фаундера — главный бюджет | `planerka`, `grill-me` |
| OKR слой | `office/templates/okr.md`, blueprints |
| Fresh verifier | `verifying`, `loop`, `done-check` |
| Уроки как живая память | `lesson`, `unstuck`, `lessons.md` |

Открытые gap фиксируются issue и не прячутся в ретро.
