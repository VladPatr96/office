# Media Blueprint

## Departments

| Department | Owns | First metric |
|---|---|---|
| Editorial | topics, voice, calendar | approved content plan |
| Production | scripts, assets, publishing package | cycle time per piece |
| Distribution | channels, titles, metadata | reach or qualified traffic |
| Analytics | feedback, retention, experiments | learning per release |
| QA | factual checks, brand checks | corrections after publish |

## Intake

- Audience and promise.
- Channels.
- Publishing cadence.
- Approval rules for public posts.
- Source and citation requirements.

## First Epics

1. Voice and topic map.
2. Three-piece pilot backlog.
3. Publishing checklist.
4. Analytics loop and lessons format.

## Gates

- Each public artifact has approval, source notes, and rollback/edit path.
- Content lessons distinguish format, topic, and distribution learnings.
