# Lesson

Record one reusable failure pattern.

## Rule

One lesson equals one pattern:

```text
symptom -> wrong instinct -> correct move
```

## Procedure

1. Confirm the pattern with evidence, not vibes.
2. Ask for human confirmation if the lesson changes future policy.
3. Write the smallest reusable entry.
4. Link issue, commit, command, or artifact.
5. Update local `lessons.md`; promote to global/project docs only if broadly reusable.

## Output

Return the lesson entry and where it was written.

Do not turn every bug into a rule. Only record patterns likely to recur.
